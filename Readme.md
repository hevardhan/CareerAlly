## Getting Started

1. Clone the repo  
   `git clone https://github.com/yourusername/CareerAlly.git`

2. Install dependencies  
   `pip install -r requirements.txt`

3. Start chatbot  
   `ollama run llama3`

4. Run backend  
   `python manage.py runserver`

5. Run frontend  
   `npm install && npm run dev`
